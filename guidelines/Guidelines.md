# KOMPAK Apps — Guidelines

Proyek ini adalah aplikasi Vue 3 (bukan React/Figma Make).

## Cursor rules (sumber utama)

Konvensi coding dan design system ada di `.cursor/rules/`:

| File | Scope |
|------|-------|
| `kompak-project.mdc` | Stack, arsitektur, struktur folder (always apply) |
| `vue-sfc.mdc` | Pola Vue SFC, props, emits, Leaflet |
| `kompak-design.mdc` | Design tokens, UI, layout |

Baca rules tersebut sebelum menambah fitur atau komponen baru.

## Design tokens

Tokens KOMPAK didefinisikan di `src/styles/globals.css` (`--kompak-*`).

## Dokumen UI/UX

Spesifikasi produk: `src/imports/KOMPAK_Apps_Dokumen_UIUX.md`
